Principais Formatações de Texto em HTML5

Negrito / Destaque

Nesta frase há um termo em negrito feito utilizando a tag <b> (não-semântica).

Nesta frase também há um termo em destaque, porém feito com a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não-semântica).

Nesta frase temos outro termo em ênfase usando a tag <em> (semântica).

Marca / Grifo

Também podemos criar uma marcação no texto com a tag <mark>,

É possível modificar o estilo de todas as tags por meio de seletores CSS.

Texto Grande e Pequeno

Outra possibilidade é mudar o tamanho das letras, deixando-as grandes com a tag <big> ou pequenas com a tag <small>.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado com a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois com a tag <ins>.

Há também como sublinhar um texto com a tag <u> (não-semântica).

Texto Sobrescrito

Um texto sobrescrito é útil para equações matemáticas, como x2 + 3 : 2. Para isso utiliza-se a tag <sup>.

Texto Subscrito

Textos subsrcritos são úteis para representar fórmulas de elementos químicos, como H2O. Para esse tipo de texto é necessário usar a tag <sub>.
